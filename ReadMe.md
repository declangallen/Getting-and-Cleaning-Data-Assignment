## Getting and Cleaning Data Assignment

###run_analysis.R

run_analysis.R does the following;

* Combines the test and train data sets.
* Assigns the label and variable names to the data set.
* Filters the data set to look for variables that include "mean" and "std"
* Returns a tidy data set that provides the average of each of the filtered variables for each volunteers activity.

Initial step for using run_analysis.R;

* Download and unzip the file to your working directory.
* Set the unzipped folder as your working directory.
